# Aplicação

Desenvolvimento em Python (Tkinter + OpenCV) para deteção de áreas por cor (HSV), extração de contornos e transformação de perspetiva.
Inclui ferramenta de color picker, ajuste dinâmico de filtros, e análise visual em grelha tipo Aruco Marker.

## Requisitos
- Python 3.10+
- opencv-python
- numpy
- Pillow

## Execução
```bash
python main.py
```
